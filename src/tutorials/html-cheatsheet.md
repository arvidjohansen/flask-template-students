# HTML Basics

## Document Structure
```html
<!DOCTYPE html>
<html>
<head>
    <title>Document Title</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>
```

## Headings
```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<!-- ... -->
<h6>Heading 6</h6>
```

## Paragraph
```html
<p>This is a paragraph.</p>
```

## Links
```html
<a href="https://example.com">Visit Example.com</a>
```

## Lists
### Unordered List
```html
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
```

### Ordered List
```html
<ol>
    <li>First Item</li>
    <li>Second Item</li>
</ol>
```

## Images
```html
<img src="image.jpg" alt="Description">
```

## Forms
```html
<form action="/submit" method="post">
    <label for="username">Username:</label>
    <input type="text" id="username" name="username" required>

    <label for="password">Password:</label>
    <input type="password" id="password" name="password" required>

    <input type="submit" value="Submit">
</form>
```

## Tables
```html
<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
    <tr>
        <td>Row 2, Cell 1</td>
        <td>Row 2, Cell 2</td>
    </tr>
</table>
```

## Divisions
```html
<div>
    <!-- Content goes here -->
</div>
```

## Span
Works like a div but does not imply line break.
```html
<span>
    <!-- Content goes here -->
</span>
```

## Semantics
### Bold and Italic
```html
<strong>Bold Text</strong>
<em>Italic Text</em>
```

### Marked Text
```html
<mark>Highlighted Text</mark>
```

### Subscript and Superscript
```html
<p>This is <sub>subscripted</sub> and <sup>superscripted</sup> text.</p>
```

These are just the basics to get started with HTML. As you progress, you can explore more advanced elements and attributes.
